OpenFL Samples (TypeScript)
===========================

## Get Started

### Clone the repository

```bash
git clone https://github.com/openfl/openfl-samples-ts
```

### Choose a sample

```bash
cd openfl-samples-ts/features/display/DisplayingABitmap
```

### Install and start

```bash
npm install
npm start
```
